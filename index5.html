Overview

ABOUT

The development of this innovative survey mobile application integrates advanced features for question analysis and response management, making it a powerful tool for efficient survey creation, management, and data analysis. The application is built using cutting-edge technologies such as Dart/Flutter for front-end development and Firestore/Firebase for back-end services. It allows users to create, modify, and analyze surveys with ease, while providing real-time data synchronization to ensure accuracy and timely access to information.

Optimizing User Experience: BeeForm's Integrated Design and Security Measures

For the front-end, Dart/Flutter was chosen as the primary technology. Dart’s versatility and Flutter’s powerful UI-building capabilities allowed us to create a dynamic and responsive interface. With Flutter, we built custom widgets that enable users to interact smoothly with the survey application, providing a seamless experience across all platforms.

The back-end is powered by Firestore/Firebase, a robust solution for real-time data storage and user authentication. Firestore ensures that all survey data is securely stored and accessible at any time, while Firebase handles user management, including login, sign-up, and session management.

This integration between Dart/Flutter and Firebase provides a highly synchronized environment where all changes are reflected instantly, ensuring users can access and interact with the most up-to-date data. The combination of these technologies not only enhances performance but also ensures that the application is secure, scalable, and easy to maintain. This makes it an ideal solution for any organization or individual looking to gather insights through surveys efficiently and effectively.

Login and Registration Interface

This is the main user interface where users can log in, select, and start a questionnaire. For new users, a "Sign Up" option is available, allowing them to create an account before logging in. The UI is designed for ease of use and optimized for various screen sizes, ensuring a smooth experience across all devices.

Questionnaire Selection Page

This page presents users with a list of available questionnaires, allowing them to select and respond to the survey of their choice.

Response Statistics and Feedback Section

This section of the application provides users with detailed statistics on the number of responded and non-responded questions. The total number of questions and completion rate are displayed to help users track their progress. Additionally, the feedback section allows users to rate their experience on a scale of 1 to 100, encouraging them to share their opinions. This feature is designed to improve the user experience and help administrators enhance future surveys. The sleek interface and clear display of data offer a seamless and user-friendly experience.

Questionnaire Page

This is the questionnaire interface where users can respond to a series of questions based on the selected topic, in this case, "Cyber Security." For each questionnaire, multiple questions are displayed, and users can proceed step-by-step through the survey. The system automatically tracks progress, showing the current question number and total number of questions, helping users keep track of their advancement.

Profile Page

This is the application's profile page, where users are greeted upon logging in. The interface allows users to customize their profile by choosing and uploading a profile picture, adding a personal touch to their account. It is designed to be clean and user-friendly, offering easy access to important sections such as Privacy Settings and the Home page. A prominent "Sign Out" button is also available, ensuring that users can securely exit the application when needed.

Questionnaire Recap and Progress Overview

This page provides users with a summary of their responses and the overall progress of the questionnaire. It displays a progress circle, indicating the percentage of questions completed, in this case showing 33%. Below, each question is listed along with the user’s corresponding answer, offering a clear and concise overview of their input. This recap feature allows users to review their responses before final submission, ensuring accuracy and completeness in their answers.
